# Glossário de Termos

- **SPA**: Single Page Application. A página não recarrega inteira, apenas o `#appContent` muda.
- **AppRouter**: O script (`app-router.js`) responsável por gerenciar a navegação da SPA.
- **ControliD**: Marca das catracas e totens faciais utilizados no condomínio.
- **PWA**: Progressive Web App. A tecnologia usada no Portal do Morador para simular um app nativo.
- **GED**: Gestão Eletrônica de Documentos.
- **O.S.**: Ordem de Serviço (Chamados de manutenção).
- **ApiBase**: Classe PHP moderna que padroniza o tratamento de requisições e respostas JSON.
