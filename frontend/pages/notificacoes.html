Como obter as credenciais do Firebase
1
Criar o Projeto no Firebase

Acesse console.firebase.google.com, clique em Adicionar projeto, dê um nome (ex: serra-liberdade-pwa), desative o Google Analytics e clique em Criar projeto.

2
Adicionar o App Web

No painel do projeto, clique no ícone </> (Web), dê um apelido (ex: Portal Morador Web), clique em Registrar app. O Firebase exibirá o bloco firebaseConfig — copie todos os valores.

3
Gerar a Chave VAPID

Vá em ⚙️ Configurações do projeto → Cloud Messaging → Configuração da Web → Certificados push da Web e clique em Generate key pair. Copie a chave pública gerada — essa é a VAPID Key.

Atenção — FCM Server Key (Legacy)

O Google descontinuou a API Legacy do FCM. O sistema está preparado para usar a Server Key Legacy se ela ainda estiver ativa no seu projeto. Caso não esteja, acesse o Google Cloud Console → APIs e serviços → Biblioteca → pesquise Cloud Messaging API e habilite a versão Legacy. A Server Key fica em ⚙️ Configurações do projeto → Cloud Messaging → Chaves de API do servidor.

5
Atualizar os arquivos de código

Após salvar as credenciais aqui, você ainda precisa atualizar manualmente os arquivos frontend/js/pwa-portal.js e firebase-messaging-sw.js com os mesmos valores do firebaseConfig. Veja o guia completo em DOC_PWA_FIREBASE.md (Passos 5 e 6).
